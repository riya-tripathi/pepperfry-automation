# 🛒 Pepperfry Automation Project

## 📌 Overview

This project automates an end-to-end user journey on the Pepperfry e-commerce website using Selenium WebDriver. It covers navigation, filtering, product selection, cart validation, and checkout flow.

---

## 🚀 Features

* Navigate through categories and subcategories
* Apply dynamic filters
* Select product from catalogue
* Handle popups and dynamic UI elements
* Add product to cart
* Open cart and verify selected product
* Proceed to checkout

---

## 🛠 Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* Page Object Model (POM)

---

## 🧠 Key Learnings

* Handling dynamic web elements and overlays
* Writing flexible XPath strategies
* Synchronization using WebDriverWait
* Handling multiple windows/tabs
* Real-world e-commerce automation

---

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/riya-tripathi/pepperfry-automation.git

2. Open in IntelliJ IDEA

3. Install Maven dependencies

4. Run:
   PepperfryTest.java

## 📷 Output

* Product successfully added to cart
* Cart verified
* Checkout page opened
